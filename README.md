# Lane Runner

A simple browser-based "subway surfer"-style 3-lane runner game. Dodge obstacles, collect coins, and beat your high score.

## Play

Live demo: `https://<your-username>.github.io/lane-runner/`

## Controls

- **Desktop**: Arrow keys or A/D to switch lanes, Up/W (or Space) to jump, Down/S to slide.
- **Mobile**: Swipe left/right to switch lanes, swipe up (or tap) to jump, swipe down to slide.

## Run locally

Just open `index.html` in any modern browser — no build step, no dependencies.

## Tech

Single-file HTML/CSS/JavaScript using the Canvas API. High score is saved in `localStorage`.

## Deploy

Hosted via GitHub Pages: Settings → Pages → Deploy from branch `main` / root.
